Correlations Discovered Between Health Risks and Poverty

Without doing any calculations you can see that there is a slight coorelation between the poverty and access to healthcare. If you click on the bubble with the state abbreviations you can see that Mississippi's poverty rate is 21.5% and lacking healthcare rate is 18.8%. An interesting outlier is Texas you can see on that graph that the poverty rate is 17.2% but 24.9% of the population lacks healthcare! Another interesting point is Massachusetts has the smallest amount of population that lacks healthcare (4.6%) but not the lowest poverty rate (11.6%).
